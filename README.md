# time-series
Report 
This report analyses Singapore Total Fertility Rate (TFR) and Total Live-Births (TLB) using annual data from 1960 to 2024

The EDA: 
data description 
data cleaning 
time plots 
analysis of time series features
first differencing 
non-seasonal ARIMA models 
AIC comparison 
residual diagnastics 

Files 
"BirthsAndFertilityRatesAnnual.csv" - raw data 
"EDA.Rmd" - main file for EDA 
"rscript" - r code 
"EDA.pdf" - knitted report output
